# Heat-SEIR-model
Temperature-driven SEIR model for heat-related disease dynamics in Dhaka, Bangladesh
#In the Heat-SEIR Model Analysis file, there a codes and figures for
  -Local and global sensitivity analysis
  -Heat SEIR Model dynamics
#Modified Heat-SEIR Model Analysis file includes
  -Model formulaton T1 and T2
  -Local sensitivity analysis
  -Gobal sensitivity analysis(PRCC)
  -Time series of R0
  -Seasonal variation of R0
  -Representation of Heat dynamics
  -Impact of transmission rate, progression rate, recovery rate
#In Data relation file, we have parameter relationships analysis
  -Max temperature and relative humidity
  -Max temperature and max dewpoint
  -Max temperature and max wind speed
  -Max temperature and surface presssure
  -Max temperature and heat index
